# Design Document - Homomorphic Filtering Project

## Project Objective

Evaluate how effectively homomorphic filtering can correct non-uniform illumination, and study how filter design choices (Butterworth vs Gaussian, parameter settings) influence the balance between illumination suppression and detail preservation.

## Theory Summary

- Image model: `f(x, y) = i(x, y) * r(x, y)`
- Pipeline: `ln -> DFT -> H(u, v) filter -> IDFT -> exp`
- Filter parameters: `gamma_L` (< 1, compresses illumination), `gamma_H` (> 1, boosts reflectance), `D0` (cutoff frequency), `c` (sharpness, Butterworth only), `n` (filter order, Butterworth only)

## Parameter Ranges to Test

- `gamma_L`: `[0.3, 0.5, 0.7, 0.9]`
- `gamma_H`: `[1.2, 1.5, 1.8, 2.0, 2.5]`
- `D0`: `[10, 20, 30, 50, 80]`
- Filter types: Butterworth (order 2), Gaussian
- For Butterworth: also vary order `n` in `[1, 2, 4]`

## Evaluation Criteria

### Qualitative (all images)

- Reduction of illumination non-uniformity
- Visibility of details in dark and bright regions
- Preservation of edges and textures
- Avoidance of over-enhancement, halo artifacts, and unnatural appearance
- Histogram shape before vs after

### Quantitative (synthetic image only)

- PSNR (Peak Signal-to-Noise Ratio)
- SSIM (Structural Similarity Index)
